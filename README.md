# 视频截图工具 (Video Screenshot)

**版本：1.0.0**

这是一个浏览器扩展，可以帮助您从各种流媒体网站获取高质量的视频截图，无需看到播放器控件或周围的网页内容。

## 主要功能

- **智能视频检测**：自动识别页面中的最佳视频元素，支持评分机制（按尺寸、加载状态、播放状态排序）
- **多平台支持**：YouTube、Netflix、Vimeo、Amazon、Hulu、HBO Max / Max、Disney+、Bilibili 等
- **多种输出格式**：PNG、JPG、WebP，或一键复制到剪贴板
- **自定义保存路径**：支持设置下载子文件夹，支持自动保存（不弹对话框）
- **快捷键支持**：默认 `Alt+S`，可前往 Chrome 扩展快捷键页面自定义
- **智能截图按钮**：半透明相机图标位于视频右上角，鼠标悬停时更明显，不影响观看体验
- **全屏模式支持**：截图按钮在全屏播放时自动适配位置
- **截图反馈**：成功时按钮播放脉冲动画，顶部滑出通知提示
- **受保护视频检测**：遇到 DRM 或受浏览器安全策略保护的视频时，给出明确提示

## 安装方法

1. 下载或克隆本仓库
2. 在 Chrome 浏览器地址栏输入 `chrome://extensions/`
3. 打开右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

首次安装后会自动打开欢迎页面，引导您快速上手。

## 使用方法

1. 播放视频时，将鼠标移到视频区域，右上角会出现半透明相机图标
2. 点击相机图标，或使用快捷键（默认 `Alt+S`）截取当前画面
3. 截图将根据您的设置自动下载或复制到剪贴板
4. 点击浏览器工具栏上的扩展图标可更改格式、保存路径和快捷键

## 设置选项

| 选项 | 说明 |
|------|------|
| 文件类型 | PNG / JPG / WebP / 复制到剪贴板 |
| 保存位置 | 相对于下载文件夹的子路径，例如 `视频截图/网站名称` |
| 自动保存 | 勾选后不弹出「另存为」对话框，直接保存 |
| 快捷键 | 点击「修改快捷键」跳转至 Chrome 扩展快捷键设置页面 |

## 技术特性

- **Manifest V3**：基于 Chrome 扩展最新架构
- `chrome.storage.sync`：设置跨设备同步，浏览器重启后持久保留
- 多 Frame 注册表：支持 iframe 内嵌视频的智能选择和状态追踪
- ResizeObserver + MutationObserver：动态适配页面变化和视频尺寸变化
- 平台适配：针对 YouTube、Bilibili 等站点的 DOM 结构做了特殊容器定位

## 项目结构

```
VideoScreenshot/
├── manifest.json        # 扩展配置（版本 1.0.0）
├── popup.html           # 扩展弹出窗口
├── firstrun.html        # 首次安装欢迎页
├── js/
│   ├── content.js       # 内容脚本：视频检测、按钮注入、截图核心
│   ├── background.js    # Service Worker：消息路由、下载、frame 注册表
│   ├── popup.js         # 弹出窗口逻辑
│   ├── settings.js      # 设置页面（备用）
│   └── firstrun.js      # 欢迎页脚本
├── css/
│   ├── style.css        # 内容脚本样式（按钮、通知）
│   └── settings.css     # 弹出窗口样式
└── icons/               # 扩展图标（16px / 48px / 128px）
```

## 注意事项

- 部分流媒体网站（如 Netflix、Disney+）使用 DRM 保护，浏览器会阻止 canvas 读取视频帧，此时插件会提示「该视频源受浏览器保护，无法直接截图」
- 截图分辨率取决于视频原始分辨率，而非播放器窗口大小
